# One-Click Dev 原型预览

## 📋 概述

这是一个完整的One-Click Dev平台HTML原型，展示了产品的UI/UX设计、交互流程和核心功能。

## 🚀 快速开始

### 方法1: 直接打开HTML文件
```bash
# 在浏览器中直接打开
open one-click-dev-prototype.html
# 或双击文件
```

### 方法2: 使用本地服务器（推荐）
```bash
# 1. 确保在prototype目录
cd /Users/k2/.openclaw/workspace/autocraft/one-click-dev/prototype

# 2. 启动预览服务器
./start-preview.sh
# 或
node preview-server.js

# 3. 在浏览器中访问
# http://localhost:3000
```

### 方法3: 使用Python简单服务器
```bash
# Python 3
python3 -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```
